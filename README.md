Клиент-серверное приложение предоставляет возможность людям с определенными пищевыми ограничениями, 
найти и выбрать продукты в онлайн-магазинах с помощью различных фильтров, таких как без глютена, без сахара, без лактозы, по различным диетам и т.д.  
Также в приложении реализована функция сканирования штрих-кода продукта, с помощью который пользователь может узнать, 
подходит ли ему продукт, исходя из его пищевых особенностей.
